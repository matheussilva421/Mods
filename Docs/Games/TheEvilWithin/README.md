# The Evil Within

Documentacao tecnica do trainer para a versao Epic Games.

## Entradas

- CT principal analisada: `EvilWithin.CT`
- CT auxiliar analisada: `The Evil Within v1.1.8.CT`
- Executavel validado: `EvilWithin.exe`

## Resultado

O trainer fica em `Games/TheEvilWithin/Trainer/` e gera um pacote Cheat Deck em:

- `Games/TheEvilWithin/Trainer/release/cheat-deck/TheEvilWithin-CheatDeck.exe`

## Compatibilidade

As assinaturas usadas no perfil `the-evil-within-epic.json` foram validadas estaticamente contra o executavel Epic recebido. `No Reload` e `No Recoil` nao foram incluidos porque as assinaturas das CTs nao bateram de forma unica nessa build.
