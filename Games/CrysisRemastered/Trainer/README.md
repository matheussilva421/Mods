# Trainer do Crysis Remastered

Este projeto foi restaurado do historico do repositorio porque ele nao deveria ter sido removido.

## O que ele contem

- `crysis_remastered_basic.ct` tabela do Cheat Engine
- `GUIA_INSTALACAO.md` guia de instalacao e uso
- `steamdeck/` scripts de apoio para Steam Deck / SteamOS

## Estrutura

- `crysis_remastered_basic.ct`
- `GUIA_INSTALACAO.md`
- `steamdeck/build.lock.example`
- `steamdeck/check_build.sh`

Este projeto representa o fluxo antigo baseado em Cheat Engine + SteamTinkerLaunch. Ele nao e o mesmo formato do trainer em `.exe` usado no Crysis 2 Remastered.
